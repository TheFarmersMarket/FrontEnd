About this Project

This site and application is built with a Rails API, Angular.js front-end, using Materialize.css framework. It was built with a mobile-first mindset and is completely responsive, so it should seamlessly work on all screen sizes, large and small. Alan Matthews and Nick Ray are responsible for creating the site's API. Spencer Wyckoff is responsible for creating the front-end.
